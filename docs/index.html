Izvor i metodologija (kliknite za detalje)

Izvor: hormuztoll.com/hormuzserver_wide.php — javni JSON s ~1290 plovila (MMSI, ime, pozicija, SOG, COG). Zastava se izvodi iz MMSI MID (prve 3 znamenke).

  • U tjesnacu — trenutna pozicija unutar 26°–27°N, 56°–57°E.
  • Namjera — u zapadnom (Perzijski zaljev) ili istočnom (Omanski zaljev) prilazu, SOG > 1 kn, COG unutar 45° od smjera prema središtu tjesnaca (26.5°N, 56.5°E).
  • U blizini — u širem boundingu (22°–28°N, 50°–62°E) bez klasifikacije gore.

Ograničenja statičke verzije: bez historijske akumulacije (trenutni snapshot). Tip broda i destinacija nisu dostupni u ovom izvoru. Puni server (GitHub repo) uz Node pokreće proširenu verziju s AIS static data.
